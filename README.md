# PsiquiatrIApp

Sistema integral para la gestión de pacientes de psiquiatría.

## Características

### Portal del Paciente
- Registro y autenticación
- Calendario interactivo para agendar citas en horarios válidos
- Visualización de recetas médicas
- Seguimiento de recomendaciones con pasos completables
- Sistema de pagos (tarjeta, transferencia, efectivo)

### Portal del Doctor
- Dashboard con estadísticas
- Calendario de citas (vista mensual y semanal)
- Bitácora de pacientes (estado de ánimo, síntomas, tratamiento, progreso)
- Creación de recetas médicas y recomendaciones

### Tecnologías
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- PWA (Progressive Web App) para soporte móvil

## Inicio rápido

```bash
npm install
npm run dev
```

Abre http://localhost:3000
